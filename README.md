# 🌾 Punjab Stubble Burning Analysis (2018–2021)

## 📌 Project Overview

This project analyzes stubble burning incidents across Punjab from **2018 to 2021** using Python, SQL, and exploratory data analysis (EDA). The objective is to identify temporal and geographical patterns, generate actionable insights, and demonstrate a complete data analytics workflow from data cleaning to storytelling.

---

## 🎯 Objective

* Clean and preprocess the raw dataset.
* Perform exploratory data analysis (EDA).
* Answer key business and environmental questions.
* Generate meaningful insights through visualizations.
* Demonstrate SQL analysis using MySQL.
* Support data-driven decision-making through storytelling.

---

## 📂 Dataset

**Source:** Reap Benefit – Climate Data Challenge

The dataset contains satellite-detected stubble burning incidents across Punjab with information such as:

* Year
* District
* Block
* Date & Time
* Day/Night
* Fire Power (W/m²)
* Latitude & Longitude
* Village (Graama)

## 📈 Business Questions Answered

* How many incidents occurred each year?
* Which districts recorded the highest number of incidents?
* Which blocks recorded the highest number of incidents?
* Were incidents more common during the day or night?
* Which months recorded the highest number of incidents?
* Which districts recorded the highest average fire power?

---

## 🔍 Key Insights

* Stubble burning incidents were concentrated in a few districts.
* Peak burning activity occurred during the post-harvest season.
* Most incidents were detected during daytime.
* Certain districts recorded higher average fire power, indicating more intense burning events.
* The analysis highlights regions that may benefit from targeted awareness and monitoring initiatives.

---

## 💡 Recommendations

* Increase monitoring in high-incidence districts.
* Conduct awareness campaigns before peak burning months.
* Encourage sustainable crop residue management practices.
* Strengthen satellite-based monitoring during high-risk periods.
